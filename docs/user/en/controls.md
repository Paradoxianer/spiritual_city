# 🕹️ Controls

SpiritWorld City supports **keyboard + mouse** on desktop/web and **touch joystick** on mobile devices.

## Keyboard Controls

### Movement (both worlds)

| Key | Action |
| :--- | :--- |
| `W`, `A`, `S`, `D` | Move up, left, down, right |
| `↑`, `←`, `↓`, `→` | Alternatively: move with arrow keys |

### Interaction — Real World

| Key | Action |
| :--- | :--- |
| `E` | Opens the **radial menu** near NPCs or buildings |
| `1` – `6` | Select action in menu, NPC dialog, or building overlay |
| `Esc` | Close menu or dialog |

### World Toggle

| Key | Action |
| :--- | :--- |
| `Q` | Toggles between **Real World** and **Spiritual World** (Costs Faith) |

### Prayer Combat — Spiritual World

| Key | Action |
| :--- | :--- |
| `1` – `4` | Choose **Prayer Mode** (see below) |
| `Space` (hold) | **Charge** prayer (watch the faith pulse) |
| `Space` (release)| **Release** prayer strike |

## Prayer Modes

During prayer combat, you have four different modes:

1.  **Liberation (🕊️ - Key 1)**: Classic prayer. Cleanses the area and deals damage to daemons.
2.  **Rebuke (🫸 - Key 2)**: Knocks daemons back and interrupts their attack.
3.  **Slow (⏳ - Key 3)**: Slows down all daemons in a large radius.
4.  **Drain (🪫 - Key 4)**: Drains energy from daemons and restores a bit of your own Faith.

## Timing is Everything

Holding `Space` charges a bar in the HUD.
*   **Optimal (70-100%)**: Maximum effect. Watch the green area and the HUD indicator!
*   **Too early/late**: Your prayer strike is much weaker.
