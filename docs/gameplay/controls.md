# 🕹️ Controls — SpiritWorld City

> **[🇩🇪 Deutsche Version weiter unten](#-steuerung--deutsche-version)**

---

## 🇬🇧 English

SpiritWorld City supports **keyboard + mouse** on desktop/web and **touch joystick** on mobile.
An in-game shortcut overlay is always available — press **F1** or **?** during gameplay.

### Keyboard Controls

#### Movement (both worlds)

| Key | Action |
|-----|--------|
| `W` / `↑` | Move up |
| `S` / `↓` | Move down |
| `A` / `←` | Move left |
| `D` / `→` | Move right |

#### Interaction — Real World

| Key | Action |
|-----|--------|
| `E` | Open the radial interaction menu (also closes it) |
| `Space` | Open radial menu / start prayer charge (context-sensitive) |
| `1` – `6` | Select action in radial menu, NPC dialog, or building overlay |
| `Esc` | Close current menu, dialog, or overlay |

#### World Toggle

| Key | Action |
|-----|--------|
| `Q` | Toggle between **Real World** and **Spiritual World** |

#### Spiritual World / Prayer Combat

| Key | Action |
|-----|--------|
| `Space` *(hold)* | Charge prayer intensity (faith pulse 0 → 100 → 0 …) |
| `Space` *(release)* | Release prayer strike at current charge level |
| `Shift` *(hold)* | Grow the prayer zone outward |
| `W` / `A` / `S` / `D` | Move **and** shape the prayer zone in that direction |

#### Help & UI

| Key | Action |
|-----|--------|
| `F1` or `?` | Toggle in-game keyboard-shortcut overlay |
| `Esc` | Close overlay / go back |

### Quick-Select Numbers (1–6) — Context Rules

The digit keys are **context-sensitive**:

| Context | What 1–6 does |
|---------|---------------|
| NPC dialog open | Select dialog action (Talk / Pray / Help / Convert …) |
| Building overlay open | Select building action (Read Bible / Pray / Worship …) |
| No dialog/overlay | Select radial menu action |

### Prayer Combat Timing

When charging prayer (`Space` held in the spiritual world):

| Charge level | Timing multiplier | Result |
|-------------|-------------------|--------|
| **70 – 100 %** | **1.0×** (optimal) | Full effect |
| 50 – 69 % | 0.8× (early) | Reduced effect |
| < 50 % | 0.6× (too early) | Weak effect |
| 0 – 30 % | 0.4× (too late) | Very weak effect |

> 💡 The HUD shows a **green optimal window** indicator — release Space when it lights up!

### Mobile / Touch Controls

On Android and iOS the game uses an on-screen joystick:

| Control | Action |
|---------|--------|
| **Left joystick** | Move the pastor / shape the prayer zone direction |
| **Action button** (bottom-right) | Interact / open menu / charge prayer |
| **Tap on NPC / building** | Open interaction when in range |

## 🇩🇪 Steuerung – Deutsche Version

SpiritWorld City unterstützt **Tastatur + Maus** am Desktop/Web und **Touch-Joystick** auf Mobilgeräten.
Ein Tastenkürzel-Overlay ist jederzeit im Spiel verfügbar — **F1** oder **?** drücken.

### Tastatursteuerung

#### Bewegung (beide Welten)

| Taste | Aktion |
|-------|--------|
| `W` / `↑` | Nach oben bewegen |
| `S` / `↓` | Nach unten bewegen |
| `A` / `←` | Nach links bewegen |
| `D` / `→` | Nach rechts bewegen |

#### Interaktion — Reale Welt

| Taste | Aktion |
|-------|--------|
| `E` | Radialmenü öffnen / schließen |
| `Leertaste` | Radialmenü öffnen / Gebet aufladen (kontextabhängig) |
| `1` – `6` | Aktion im Radialmenü, NPC-Dialog oder Gebäude-Overlay auswählen |
| `Esc` | Aktuelles Menü, Dialog oder Overlay schließen |

#### Weltenwechsel

| Taste | Aktion |
|-------|--------|
| `Q` | Zwischen **Realer Welt** und **Geistlicher Welt** wechseln |

#### Geistliche Welt / Gebetskampf

| Taste | Aktion |
|-------|--------|
| `Leertaste` *(halten)* | Gebetsintensität aufladen (Glaubenspuls 0 → 100 → 0 …) |
| `Leertaste` *(loslassen)* | Gebetsschlag mit aktuellem Ladestand auslösen |
| `Shift` *(halten)* | Gebetszone nach außen vergrößern |
| `W` / `A` / `S` / `D` | Bewegen **und** Gebetszone in diese Richtung formen |

#### Hilfe & UI

| Taste | Aktion |
|-------|--------|
| `F1` oder `?` | Tastenkürzel-Overlay ein-/ausblenden |
| `Esc` | Overlay schließen / zurück |

### Schnellauswahl-Ziffern (1–6) — Kontextregeln

Die Zifferntasten sind **kontextabhängig**:

| Kontext | Was 1–6 macht |
|---------|---------------|
| NPC-Dialog geöffnet | Dialog-Aktion auswählen (Sprechen / Beten / Dienen / Bekehren …) |
| Gebäude-Overlay geöffnet | Gebäude-Aktion auswählen (Bibel lesen / Beten / Gottesdienst …) |
| Kein Dialog/Overlay | Radialmenü-Aktion auswählen |

### Gebetskampf-Timing

Beim Aufladen des Gebets (`Leertaste` halten in der geistlichen Welt):

| Ladestand | Timing-Multiplikator | Ergebnis |
|-----------|----------------------|----------|
| **70 – 100 %** | **1,0×** (optimal) | Volle Wirkung |
| 50 – 69 % | 0,8× (zu früh) | Reduzierte Wirkung |
| < 50 % | 0,6× (zu früh) | Schwache Wirkung |
| 0 – 30 % | 0,4× (zu spät) | Sehr schwache Wirkung |

> 💡 Im HUD wird ein **grünes Optimal-Fenster** angezeigt — Leertaste loslassen, wenn es leuchtet!

### Mobile / Touch-Steuerung

Auf Android und iOS gibt es einen On-Screen-Joystick:

| Steuerung | Aktion |
|-----------|--------|
| **Linker Joystick** | Pastor bewegen / Gebetszone formen |
| **Aktionsknopf** (unten rechts) | Interagieren / Menü öffnen / Gebet aufladen |
| **Tippen auf NPC / Gebäude** | Interaktion öffnen (wenn in Reichweite) |
